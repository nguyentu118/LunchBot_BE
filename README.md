# 🍱 LunchBot – Online Food Ordering System (Backend)

## 📌 Overview
LunchBot is a backend system for an online food ordering platform.  
It provides RESTful APIs for managing users, menus, carts, and orders with secure authentication.

This project demonstrates backend development skills using Spring Boot, JWT authentication, and real-time communication.

---

## 🛠 Tech Stack
- Java 17
- Spring Boot
- Spring Security (JWT)
- MySQL
- WebSocket
- Maven

---

## 🚀 Features
- User authentication & authorization with JWT
- Menu and product management
- Shopping cart functionality
- Order creation and tracking
- Real-time order notifications using WebSocket
- Payment integration (SePay)

---

## 🧩 Architecture
The application follows a layered architecture:

Controller → Service → Repository → Database

This structure helps separate concerns and makes the project easier to maintain and scale.
